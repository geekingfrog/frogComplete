FrogComplete

A small autocomplete widget.

Super simple to use

 
  var data = ["blue", "green", "red", "orange", "purple", "yellow"];
  new Autocomplete('input#colors', data);

Try entering some colors

Works with complex data

 
  var timezones = [{"cc":"TW", "offset":"+08:00,+08:00", "city":"Taipei"}, (...)];
  new Autocomplete('input#timezones', timezones, {
    value: function(datum) { return datum.city; }
  });

Enter a city for a timezone.

Custom display for suggestions

 
  var options = {
    value: function(d) { return d.city; },
    display: function(datum, input) {
      var transform = new RegExp("(" + input + ")", "i");
      return datum.city.replace(transform, "<span style='color: red;'>$1</span>");
    }
  };
  new Autocomplete('input#timezones', timezones, options);

Enter a timezone.

Built-in validation

 
  var validateAutocomplete = new Autocomplete('input#timezones3', timezones, {
    value: function(d) { return d.city; },
    validation: true
  });

test the validity of the input:

 
  validateAutocomplete.getIsValid(); //true if the user selected a timezone

This form will submit only if you click on one of the suggestion.
